## need  to have installed 

1. Install docker

## Run project

1. git clone https://github.com/EduardoHead18/save---producs.git
2. configure the .env.example file (You can review the postgres documentation to know   how to configure the connection).
3. Run: 
    ```bash
    docker-compose up -d
    ```
